# Core Visual Traits

This document catalogs the visual properties common across fractal families. These are observable, describable features used for classification, prompting, shader design, and visual taxonomy.

---

## Structural Traits

### Self-Similarity
The defining visual property of fractals: sub-regions of the image resemble the whole, either exactly or approximately. Observable at multiple zoom levels. Examples:
- Mandelbrot Set: embedded "minibrots" in filament regions
- Sierpinski Triangle: each sub-triangle is a smaller copy of the whole
- Koch Snowflake: each edge segment has the same zigzag profile as the whole curve

### Infinite Detail
Zooming into a fractal boundary or structure continues to reveal new complexity. There is no "bottom level" at which the image becomes smooth or featureless. This is a direct consequence of the mathematical structure, not a rendering artifact.

### Branching
Hierarchical division into sub-branches, each of which further divides. Seen in:
- Fractal trees (binary, ternary, random-angle)
- Barnsley Fern (leaflet repetition)
- Mandelbrot filaments (antenna structures with dendritic branching)
- Vascular-like IFS structures

### Filaments and Tendrils
Thin, thread-like structures extending from denser regions. Characteristic of escape-time fractals. The Mandelbrot Set has antenna-like filaments extending from its boundary. Julia sets near the boundary of the Mandelbrot Set become entirely filamentary (Dendrite Julia sets).

### Bulbs and Lobes
Rounded enclosed regions, typically connected to each other and to a main body. The Mandelbrot Set's cardioid and period-2 bulb are canonical examples. Each bulb corresponds to a specific periodic orbit.

### Dust and Disconnected Point Sets
Fractal sets that consist of infinitely many points with no connecting arcs. The Cantor Set is the canonical example. Julia sets for parameter values outside the Mandelbrot Set become "Cantor dust" — totally disconnected point clouds.

### Space-Filling Behavior
Some fractal curves pass through or arbitrarily closely approach every point in a 2D region. Hilbert Curve, Peano Curve, and Gosper Curve are space-filling: their Hausdorff dimension is 2 despite being generated by 1D curves.

## Boundary and Edge Properties

### Dense, Complex Boundaries
The boundary of the Mandelbrot Set has Hausdorff dimension 2 — it is as "thick" as a 2D region despite being a 1D boundary. Fractal boundaries in general are far more complex than smooth Euclidean boundaries.

### Fjord Structures
Deep, narrow inlets into the main body of a fractal set. Visible in the Mandelbrot Set as the thin channels connecting bulbs to the main cardioid.

### Satellite Structures
Smaller copies of the main form attached to or embedded in the boundary at precise parameter values. Every point on the Mandelbrot boundary is a limit of embedded minibrots.

---

## Scale and Repetition Properties

### Scale Invariance
Visual patterns that repeat across zoom levels without a preferred scale. No matter how far you zoom into a self-similar fractal, the image looks statistically equivalent.

### Nested Structure
Structures within structures within structures: bulbs contain smaller bulbs, branching patterns contain sub-branches, spiral arms contain sub-spirals.

### Periodic Motifs
In escape-time fractals, different regions of parameter space produce orbits of different periods. These periods are visible as distinct visual territories (period-1 = cardioid, period-2 = main bulb, period-3 = small bulbs, etc.).

---

## Texture and Density Properties

### Lacunarity
The "gappiness" of a fractal — how much void space is distributed relative to filled space. High lacunarity: large gaps with dense clusters. Low lacunarity: uniform distribution. The Cantor Set has high lacunarity; the Sierpinski Carpet has lower lacunarity than Cantor dust.

### Fill vs. Void Contrast
The ratio of filled (interior) to void (exterior) space, and how that ratio scales with observation scale. IFS attractors tend toward high fill; Cantor-type sets tend toward high void.

### Edge Density
How much boundary per unit area exists within a region. Mandelbrot Set boundary regions have extremely high edge density. Interior cardioid regions have zero edge density.

---

## Motion and Animation Traits

### Zoom Self-Similarity
Under continuous magnification, a self-similar fractal passes through repeating visual states. Spiral zooms through the Mandelbrot Set return to visually equivalent configurations at regular (in log scale) intervals.

### Parameter Animation Behavior
Small changes in iteration parameters produce smooth visual changes in some regions (interior coloring) and discontinuous structural changes in others (near bifurcation points). Animation through parameter space reveals the fractal structure of parameter space itself.

### Escape-Time Gradient Banding
When coloring by escape iteration count, the boundaries between color bands are themselves fractal curves, producing nested color halos around the fractal boundary.
